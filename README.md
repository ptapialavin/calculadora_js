# calculadora_js
🖥️ Tarjeta de Producto Interactiva - Laptop Gamer AMD
📌 Descripción
Proyecto desarrollado como parte del curso Desarrollo Full Stack JavaScript de Desafío Latam, en el módulo JavaScript para la Web, unidad Introducción a JavaScript.
La aplicación consiste en una tarjeta de producto que permite aumentar o disminuir la cantidad de una Laptop Gamer AMD, calculando automáticamente el total a pagar.
🚀 Tecnologías

HTML5
CSS3
JavaScript (Vanilla JS)

✨ Funcionalidades

Mostrar precio base del producto.
Incrementar y disminuir la cantidad de productos.
Evitar cantidades negativas.
Actualizar el total de compra en tiempo real.

🧠 Conceptos aplicados

Manipulación del DOM (querySelector).
Eventos (onclick).
Variables y operadores aritméticos.
Actualización dinámica de contenido (innerHTML).
Uso de funciones.
Control de límites con Math.max().
